$ brew install mongodb

create the database in ./data/db

start mongo daemon
$ cd <main_project_dir>/data
$ mongod --dbpath ./db


start mongodb shell and create a database for test
$ mongo --host 127.0.0.1:27017

this command creates the database "picinfo"
> use picinfo
